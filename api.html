Plugin API

Bukkit API by EssentialsKT

EssentialsKT is powered by a flexible and powerful in-house plugin API. This page details the variety of features offered by the EssentialsKT Plugin API.

JavaDoc

Package pluginapi.chat

ClickableText.kt

  • Follows Java builder pattern
  • Design your clickable text component
  • Supports any ClickEvent.Action-value pair: COPY_TO_CLIPBOARD, RUN_COMMAND, SUGGEST_COMMAND, OPEN_URL, etc.
  • Easily translates to ChatComponentAPI TextComponent via the #toComponent() method

HoverableText.kt

  • Follows Java builder pattern
  • Design your hoverable text component
  • Display multi-line text on hover over the component
  • Easily translates to ChatComponentAPI TextComponent via the #toComponent() method
