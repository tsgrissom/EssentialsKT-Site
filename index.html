EssentialsKT is a Minecraft plugin project by T0xicTyler

For CraftBukkit servers running EssentialsX
Developed targeting modern Java 17, written in Kotlin

The core plugin is an addon for EssentialsX which provides many new commands and additional features for existing commands, in addition to features missing from the base Essentials experience.

Further, EssentialsKT is backed by a powerful in-house plugin API, the features of which are outlined below. For more information, see the Plugin API page.

Projects

EssentialsKT

Addon for EssentialsX Minecraft plugin

  • Many new commands, all matching the style of Essentials
    • Greater command intelligence
    • Superior /gamemode + /gmx commands: GUI options, fancy text,
    • Enhanced /whoiswith fancy text
    • Admin commands with copy-paste support: /ipaddress, /uuid, etc.
    • Quick time setting commands
    • Quick weather setting commands + restoration of /toggledownfall
    • See all commands
  • Permission structure derived from that of EssentialsX to ensure minimal breakage
  • Missing features of EssentialsX such as join and leave messages
  • Configurable plugin features

Plugin API

Powerful plugin API designed to be used with Kotlin

  • Chat Component API builders for clickable and hover-able text
  • An extensive Command API including:
    • Powerful Terminal-style command flag parsing
    • Command help + usage generation including fancy text
    • CommandExecutor and parameter wrappers with tons of functionality
  • Minecraft-relevant Kotlin extensions including:
    • Boolean formatting
    • Numeric formatting
    • String functions
    • Bukkit CommandSender, ComponentBuilder, ItemStack, Player, & World functions
